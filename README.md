# IYEON Enterprise Platform OS v3.0.0

Next.js + TypeScript + Tailwind CSS + Supabase 기반 이연 엔터프라이즈 플랫폼 Foundation입니다.

## 포함 내용
- src 기반 프로젝트 구조
- 이연 오렌지 브랜드 컬러 시스템
- Header / Footer / Navigation
- 공통 UI: Button, Container, Section
- 기본 Home Foundation 화면
- Google / Naver site verification metadata 유지
- Supabase client 기본 준비

## GitHub 업로드
이 폴더 안의 모든 파일과 폴더를 `iyeon-enterprise-platform` 저장소에 업로드하세요.

권장 커밋 메시지:

```bash
feat: initialize IYEON Enterprise Platform OS v3 foundation
```

## Vercel 환경변수
Supabase 연결 시 아래 값을 Vercel Environment Variables에 입력합니다.

```bash
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
```
